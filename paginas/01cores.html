Cores

Sites para ideias de cores

Há vários sites que nos possibilitam essa funcionalidade, porém, nesse resumo vou exemplificar apenas algumas delas.

Adobe color

Site oficial: https://color.adobe.com
Nesse site você encontra:

  • Ideia de paleta de cores;
  • Extrair cores a partir de uma imagem;
  • Montar um degradê com as cores que quiser;
  • Sugestões de cores e paletas;
  • Tedências de cores para determinados assuntos.

Paletton

Site oficial: https://paletton.com
Nesse site você encontra:

  • Ideia de paleta de cores;
  • Te mostra uma simulação de um site com as cores escolhidas.

Coolors

Site oficial: https://coolors.co
Nesse site você encontra:

  • Sugestão de paletas de cores;
  • A cada vez que pressionar barra de espaço te sugere uma paleta diferente.

Obter cores de sites da Web

ColorZilla

É uma extenção do google que possibilida a obtenção do código hexadecimal da cor de um site na Web.
Link de download: https://chrome.google.com/webstore
